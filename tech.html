Code & Repositories

Open code, open hardware.

All project code is open on GitHub. The firmware, the Python bridge, the classifier training pipeline, and the Flutter companion app each live in their own repo.

C++ · Arduino

Sleeve Firmware

Adafruit Feather nRF52840 firmware: I²C mux drivers, IMU/EMG/PPG sampling loop, USB serial streaming.

github.com/team-r/sleeve-firmware
Python · scikit-learn

ML Pipeline

Feature extraction, Random Forest training, GroupKFold cross-validation, and confusion matrix generation.

github.com/team-r/sleeve-ml
Python · WebSockets

Serial Bridge

Reads serial packets from the MCU and broadcasts to the Flutter app over WebSocket for real-time inference.

github.com/team-r/sleeve-bridge
Dart · Flutter

Companion App

iOS Flutter app: live form feedback, rep counting, session history backed by Supabase.

github.com/team-r/sleeve-app
