Engineering Specifications

System parameters were selected to balance signal quality, stable multi-sensor streaming, and real-time responsiveness.

Core Technical Specs

Parameter Selected Value Purpose
EMG Sampling Rate 1000 Hz Captures rapid muscle activation dynamics.
IMU Sampling Rate ~60 Hz Supports real-time upper-limb motion tracking.
PPG Update Rate ~1 Hz Appropriate for slower heart-rate variation.
Serial Baud Rate 230400 baud Improves stability for wired multi-sensor streaming.
ML Classifier Random Forest Classifies bicep curl form from per-repetition features.

Signal Processing

Signal Applied Filtering Purpose
EMG 20–250 Hz bandpass + envelope smoothing Reduces noise and extracts muscle activation amplitude.
IMU Angle Complementary filter + 4 Hz low-pass Stabilizes elbow angle estimation and suppresses jitter.
PPG / HR Low-pass EMA smoothing Reduces pulse noise and smooths heart-rate readings.
